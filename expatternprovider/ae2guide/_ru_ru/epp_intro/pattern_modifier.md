# Модификатор шаблонов

Модификатор шаблонов — это инструмент для пакетной модификации шаблонов.

<ItemImage id="expatternprovider:pattern_modifier" scale="4"></ItemImage>

Щёлкните [ПКМ], чтобы открыть его GUI.

## Режим умножения

Вы можете умножить/разделить входное и выходное количество шаблона обработки на x, нажав соответствующую кнопку.

![PM](../pic/pm.png)

Исходный шаблон:

![PM1](../pic/pm1.png)

После 10-кратного изменения:

![PM2](../pic/pm2.png)

Он также может очистить содержимое всех шаблонов и преобразовать их в пустые шаблоны, нажав кнопку «Очистка».

### Примечания:

 - Кнопка деления работает только при кратности количества ингредиентов. Например, кнопка ÷2 не будет работать, если шаблон требует три
булыжника на входе, поскольку 3÷2 равно 1,5.

 - Кнопка умножения имеет ограничение (999999). Она не может превысить это число для одного ингредиента.

## Режим замены

Заменить определённый входящий и исходящий ингредиент шаблона другим предметом.

![PM3](../pic/pm4.png)

Слот A — это то, что будет заменено, а слот B — то, чем будет заменен целевой элемент.

Например, следующая настройка заменит дубовые доски углём.

![PM4](../pic/pm6.png)

![PM5](../pic/pm5.png)

Нажмите на кнопку «Замена», чтобы выполнить замену.

![PM6](../pic/pm7.png)
